# Inter Bank Transfer Management System

A web-based backend web application created with Java Spring Boot built for 
Sri Lanka Insurance Corporation General Limited (SLIC General).

## Overview
This system manages and tracks inter-bank fund transfers within 
the organization, replacing manual processes with a streamlined 
digital workflow.

## Features
- Inter-bank transfer request management
- Transfer status tracking
- Finance department approval workflow
- Transaction history and reporting
- User authentication and access control

## Purpose
Built as an internal tool for the Finance Department to improve 
efficiency and accuracy of inter-bank transfer operations.
